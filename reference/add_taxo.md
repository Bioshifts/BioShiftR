# Add taxonomy to range shifts

add_taxo() merges taxonomic classification to range shifts uploaded with
the get_shifts() function.

## Usage

``` r
add_taxo(data)
```

## Arguments

- df:

  input shifts dataset from get_shifts() function.

## Value

Range shifts dataset with added columns containing taxonomic
classification information for every species name, as written in the
original publication.
