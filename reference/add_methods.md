# Add methods to range shift documentations

add_methods supplements the minimal shifts database with methodological
information describing the methods by which individal shifts were
documented. These include variables such as sampling years, definitions
of range parameters, and more.

## Usage

``` r
add_methods(data)
```

## Arguments

- data:

  input data from get_shifts().

## Value

original shifts dataset, supplemented with methodological parameters for
each individual range shift detection. See readme for details on methods
columns.
