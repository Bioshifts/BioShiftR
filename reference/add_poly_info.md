# Add polygon attributes to range shift dataframe

This function supplements range shift observations ()

## Usage

``` r
add_poly_info(data, type = "SA")
```

## Arguments

- data:

  Range shifts dataframe from
  [`get_shifts()`](https://bioshifts.github.io/BioShiftR/reference/get_shifts.md)
  function

- type:

  Specification for study area polygon info ("SA"), or species-specific
  study area polygon info ("SP")

## Value

Range shift dataframe supplemented with columns on spatial extent of
study or species-specific polygons.
